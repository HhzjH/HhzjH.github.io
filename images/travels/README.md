# Travels Images Directory

This directory contains images for your travel timeline.

## How to Add Your Travel Photos

1. Place your travel photos in this directory
2. Name them descriptively (e.g., `beijing1.jpg`, `beijing2.jpg`, `hangzhou1.jpg`, etc.)
3. Update the image references in `/content/travels.md` to match your photo filenames

## Example Structure

```
travels/
├── beijing1.jpg       # 故宫照片
├── beijing2.jpg       # 长城照片
├── hangzhou1.jpg      # 西湖照片
├── hangzhou2.jpg      # 西湖夕阳照片
├── chengdu1.jpg       # 大熊猫照片
├── chengdu2.jpg       # 宽窄巷子照片
├── guilin1.jpg        # 漓江照片
└── guilin2.jpg        # 象鼻山照片
```

## Recommended Image Specifications

- Format: JPG, PNG, or WebP
- Recommended width: 800-1200px
- File size: Keep under 500KB for optimal loading speed
- Aspect ratio: 16:9 or 4:3 works best
